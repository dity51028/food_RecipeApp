# Food Recipe API - Quick Reference

## Server Info
- **Base URL**: `http://localhost:3001`
- **Database**: MongoDB (food_recipe)

## PowerShell Commands

### 1. Get All Recipes
```powershell
Invoke-RestMethod -Uri "http://localhost:3001/recipe" -Method GET
```

### 2. Get Recipe by ID
```powershell
Invoke-RestMethod -Uri "http://localhost:3001/recipe/YOUR_RECIPE_ID" -Method GET
```

**Example:**
```powershell
Invoke-RestMethod -Uri "http://localhost:3001/recipe/697e2271f52d58932a8e3e82" -Method GET
```

### 3. Add New Recipe
```powershell
Invoke-RestMethod -Uri "http://localhost:3001/recipe/add" -Method POST -Body '{"title":"Recipe Title","ingredients":"ingredients list","instruction":"cooking instructions"}' -ContentType "application/json"
```

**Example:**
```powershell
Invoke-RestMethod -Uri "http://localhost:3001/recipe/add" -Method POST -Body '{"title":"Chicken Curry","ingredients":"500g chicken, 2 onions, garlic, curry powder, coconut milk","instruction":"Cook chicken. Saut√© onions. Add spices and coconut milk. Simmer for 20 minutes."}' -ContentType "application/json"
```

### 4. Update Recipe
```powershell
Invoke-RestMethod -Uri "http://localhost:3001/recipe/YOUR_RECIPE_ID" -Method PUT -Body '{"title":"Updated Title","ingredients":"updated ingredients","instruction":"updated instructions"}' -ContentType "application/json"
```

### 5. Delete Recipe
```powershell
Invoke-RestMethod -Uri "http://localhost:3001/recipe/YOUR_RECIPE_ID" -Method DELETE
```

## Browser Access

### View All Recipes
Open in browser: `http://localhost:3001/recipe`

### View Specific Recipe
Open in browser: `http://localhost:3001/recipe/YOUR_RECIPE_ID`

## Recipe Data Structure

**Required Fields:**
- `title` (string)
- `ingredients` (string) 
- `instruction` (string)

**Optional Fields:**
- `time` (string)
- `coverImage` (string)

**Example Recipe Object:**
```json
{
  "title": "Spaghetti Carbonara",
  "ingredients": "400g spaghetti, 200g bacon, 4 eggs, 100g parmesan",
  "instruction": "Cook pasta al dente. Fry crispy bacon. Mix eggs and cheese. Combine with seasoning.",
  "time": "30 minutes",
  "coverImage": "https://example.com/image.jpg"
}
```

## Error Responses

- **400 Bad Request**: Missing required fields
- **404 Not Found**: Recipe ID not found
- **500 Internal Server Error**: Server error

## Notes
- Server runs on port 3001
- All timestamps are in UTC
- IDs are automatically generated by MongoDB
- Use the `_id` field for finding specific recipes
